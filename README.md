# porn-crawler

<h2>set up requirements</h2>
 pip install -r .\requirements.txt 
 
<h2>keep running to update database</h1>
 cd porn-crawler 
 python crawler.py 

<h2>open the server</h2>
 cd porn-crawler\website\manage.py 
 python manage.py runserver 

<h2>open the browser</h2>
 http://127.0.0.1:8000
